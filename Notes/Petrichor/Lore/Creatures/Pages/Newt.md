# <span class="lore-title-inline">Notes</span>
The [[Newt]] encountered in the [[Bazaar Between Time]] is a specific `Time Newt`.

This specific [[Newt]] has a defective left arm, leaving them unable to perform the same acts seen from other `Time Newts`. Additionally, it is mentioned that the [[Newt]] is blind. It is unclear if this is a unique feature of the [[Newt]] encountered in the [[Bazaar Between Time]], or if this is true of all `Time Newts`.

The [[Bazaar Between Time]] is run by the [[Newt]], where they can be found selling various `Lunar` items. This is due to the influence of [[Mithrix]].
