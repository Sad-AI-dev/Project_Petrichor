# <span class="lore-title-inline">Notes</span>
The [[Queen's Gland]] can be retrieved from a [[Beetle Queen]].

This log shows how the [[Queen's Gland]] is a [[Beetle Queen]] organ that attracts [[Beetle Guard|Beetle Guards]] and potentially allows the [[Beetle Queen]] to control the [[Beetle Guard|Beetle Guards]].

It is possible that without the influence of a [[Queen's Gland]], the [[Beetle Guard|Beetle Guards]] start to fight each other for dominance.
